侧边栏内容（从顶部到底部）

  1. 头部 — 渠道名称 + 关闭按钮
  2. 基本信息区 — 名称 / Base URL / API Key / 优先级 / 备注（当前 Modal 里的内容）
  3. 模型与别名区 — 当前展开区域的模型卡片 + 别名管理 + 定价
  4. 操作区 — 拉取上游模型 + 连通检测 + 保存 / 取消

交互流程

打开
点击渠道卡的"编辑"按钮 → 右侧滑出侧边栏 → 背景半透明遮罩
关闭
点击遮罩 / 关闭按钮 / Escape 键 → 保存后自动关闭刷新
模型操作
添加/删除模型、别名、定价都在侧边栏内完成，不需要弹新窗
健康检测
"连通检测"按钮也放到侧边栏底部，保持一个地方完成所有配置
